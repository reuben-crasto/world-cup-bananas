# World Cup Bananas

A FIFA World Cup 2026 prediction game. Predict group stage scores, knockout round winners, and compete with friends based on accuracy.

## Features

- User sign up and sign in
- Group stage score predictions for all 48 matches
- Auto-updating group standings tables
- Autofill options (FIFA rankings, balanced, random)
- Points awarded based on prediction accuracy

## Getting Started

### Prerequisites

- Node.js (v16+)

### Installation

```bash
npm install
```

### Running

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
